Quick Start

🚀 Development Setup

# Clone repository
git clone https://github.com/oneseco-media/ai-shortcut-companion.git

# Install dependencies
npm install

# Set environment variables
DATABASE_URL=your_postgres_url
OPENAI_API_KEY=your_api_key

# Initialize database
npm run db:push

# Start development server
npm run dev
